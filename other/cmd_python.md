#### 版本
```
安装python
有2.x和3.x两个版本，两个版本的python代码不兼容
py --version                    查看版本
py -V

安装python后，在系统中安装了一个启动器py.exe，默认放置在文件夹C:\Windows\下面。在命令行输入：
py -2 <file.py>                 使用python2运行
py -3 <file.py>                 使用python3运行
```

#### 库安装
```
python内置了pip
py -3 -m pip install <libname>  安装库到python3
```
