# IntelliJ IDEA

[返回][ide]

## 快捷键

```text
ctrl + alt + s                  Settings...(设置)

ctrl + f                        Find...(在文件中查找)
ctrl + shift + f                Find in Path...(全文查找)

ctrl + r                        Replace...(在文件中替换)
ctrl + shift + r                Replace in Path...(全文替换)

ctrl + e                        Recent Files(最近打开文件)
ctrl + shift + e                Recent Changed Files(最近修改文件)

ctrl + z                        Undo Typing(撤销)
ctrl + shift + z                Redo Typing(恢复撤销)

ctrl + n                        Class...(查找类)
ctrl + shift + n                File...(查找文件)

ctrl + alt + o                  Optimize Imports
ctrl + alt + l                  Reformat Code(格式化代码)
ctrl + alt + t                  Surround With
alt + insert                    Generate...

ctrl + p                        Parameter Info

ctrl + g                        Line/Column

ctrl + shift + -                Collapse All(折叠所有)
ctrl + -                        Collapse(折叠当前)
ctrl + shift + =                Expand All(展开所有)
ctrl + =                        Expand(展开当前)
ctrl + shift + u                Toggle Case(转变大小写)

ctrl+[                          Move caret to Code Block Start(把光标移动到块头)
ctrl+]                          Move caret to Code Block End(把光标移动到块尾)

alt + button1 click             Add or Remove Caret
alt + shift + button1 click     Create Rectangular Selection on Mouse Drag
```
